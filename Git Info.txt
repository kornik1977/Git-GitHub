1. How to push repo into GitHub?*
Step by step do this:
git init - new (empty) Repo is created - optional
git add "file url" (. adds all)
git commit -m "Initial commit"
git remote add origin <project url>  - connects local Repo with Rep on GitHub - optional
git push -f origin master
